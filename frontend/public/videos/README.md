# Pre-loaded Videos

Place your video files in this directory.

## Supported Formats
- MP4 (recommended)
- WebM
- OGG
- Any format supported by ReactPlayer

## Usage
Videos placed here can be accessed via:
- `/videos/your-video.mp4` (relative path)
- Or use the video selector in the app

## Example
If you place `sample.mp4` here, you can access it at:
`http://localhost:5173/videos/sample.mp4`
